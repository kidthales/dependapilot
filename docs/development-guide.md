# Development Guide

## Requirements

- [Docker Compose](https://docs.docker.com/compose/install/) (v2.10+)
- [Git](https://git-scm.com/)
- [Make](https://www.gnu.org/software/make/)

## Quick Start

```shell
git clone https://github.com/kidthales/dependapilot.git
cd dependapilot
make build
make up
```
